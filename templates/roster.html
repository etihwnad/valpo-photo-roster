Instructions

This form accepts a CSV file exported from the Canvas gradebook. It uses only the "Student and "SIS User ID" columns from that export. Edit the CSV file as desired to use preferred names, etc. Names are interpreted as "Last, First" since the export doesn't provide separate columns.

Get to the Canvas download form by: Grades → (button) Export → Export Entire Gradebook.

Thanks to Stewart Thomas for the nice transparent overlay formatting of the names with TikZ.

Errors or comments? Contact Dan White.
